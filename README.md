# Plataforma Web para la Validación Automática de la Legibilidad de Documentos Escaneados

## Descripción

Este proyecto corresponde al prototipo desarrollado para la tesis de grado titulada:

"Desarrollo de una plataforma web para la validación automática de la legibilidad de documentos escaneados mediante aprendizaje automático con Teachable Machine."

## Tecnologías utilizadas

- Python
- Flask
- TensorFlow
- Teachable Machine
- OpenCV
- HTML
- CSS
- JavaScript

## Funcionalidades

- Inicio de sesión
- Carga de múltiples documentos
- Conversión de PDF a imágenes
- Clasificación automática de legibilidad
- Visualización por página
- Estadísticas de documentos clasificados
